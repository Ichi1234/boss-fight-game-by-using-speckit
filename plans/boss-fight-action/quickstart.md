## Local dev (npm scaffold)

1. Install dependencies:

```powershell
npm install
```

2. Start dev server:

```powershell
npm run dev
```

3. Open `http://localhost:3000` to play the demo.

## CDN single-file demo

Open `examples/boss-fight/index.html` in a browser. The file includes Phaser via CDN and loads local assets.

## Design tokens (colors/timings)

- Primary background gradient: #0b1220 -> #243042
- Player slash color: white, duration 120ms
- Telegraph flash: 250ms pre-attack
- Dodge invulnerability: 200ms flicker
